# MS Exam Dump Prep

A React app to study for Microsoft security certifications: **SC-900**, **SC-200**, and **SC-100**.

## Features
- 3 exam tracks with 100+ questions each
- Multiple choice quizzes with explanations
- Score tracking and review mode
- Shuffle questions per session

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
